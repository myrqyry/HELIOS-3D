# User Profile: HELIOS-3D Principal Investigator

## Context
The user is the Lead Investigator for HELIOS-3D, focused on ultra-low-energy topological computing. They prefer a senior-peer level of collaboration—concise, high-signal, and proactive.

## Preferences
- **Tone:** Professional, direct, senior-engineer style. No filler.
- **Workflow:** Prefers deep research into specific physics/engineering bottlenecks followed by surgical documentation updates.
- **Tools:** Uses `NotebookLM` for deep literature querying and micromagnetic simulations (MuMax3/OOMMF) for validation.
- **Project Structure:** Highly organized Documentation (`docs/`), Simulations (`simulations/`), and Research Specs (`research_specifications/`).
- **Standard:** Adheres strictly to the `[DEMONSTRATED]` vs `[SPECULATIVE]` tagging protocol.

## Goals
1. Validate room-temperature 3D hopfion stability.
2. Resolve the "Readout Bottleneck" via microwave/magnonic sensing.
3. Scale DISH volumetric printing for coprocessor integration.
4. Establish a credible "Planar-First" baseline demonstrator.

## Learned Lessons
- User appreciates proactive identification of "weak areas" in the proposal (e.g., DISH preprocessing times).
- User wants integration across all repo levels (README, Matrix, Glossary, etc.) once a new finding is validated.
