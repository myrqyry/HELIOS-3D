# src/components

## What Lives Here

Reusable Astro and React UI, visual summaries, charts, research panes, and
Three.js client components.

## Routing

- `ui/` -> small interactive UI controls
- `r3f/` -> client-only Three.js scenes
- `research/` -> research browsing views
- `visual-summaries/` -> doc summary panels
- `charts/` -> chart and timeline components
