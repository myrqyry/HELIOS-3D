# src

## What Lives Here

Astro pages, reusable components, shared utilities, and content collections.

## Routing

- When editing pages, follow the page-specific notes below.
- `src/pages/` -> route-level page files
- `src/components/` -> reusable UI and visual systems
- `src/data/` -> research/data modules
- `src/utils/` -> pure helpers
- `src/content/docs/` -> public documentation source

## Page Notes

- Keep page logic thin and push reusable behavior into components or utils.
- Preserve route params and metadata behavior when refactoring.
