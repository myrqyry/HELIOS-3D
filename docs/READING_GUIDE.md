# Reading Guide

HELIOS-3D is a large, speculative research documentation project. This guide helps you navigate it efficiently based on your background and interests.

## Quick Start

**If you are new to the project:**
1. Read the project README for the high-level framing and status
2. Read [`ABSTRACT.md`](./ABSTRACT.md) for the full system hypothesis
3. Read [`ALTERNATIVE_MATERIALS_AND_METHODS.md`](./ALTERNATIVE_MATERIALS_AND_METHODS.md) for the current baseline path (planar-first, electrical readout)

**If you want the most defensible content:**
- Start with [`ALTERNATIVE_MATERIALS_AND_METHODS.md`](./ALTERNATIVE_MATERIALS_AND_METHODS.md) -- this is the near-term demonstrator path
- Then review [`CLAIMS_MATRIX.md`](./CLAIMS_MATRIX.md) to see which claims are demonstrated vs. speculative

**If you are a physicist/materials scientist:**
- [`CANDIDATE_MATERIALS_AND_MECHANISMS.md`](./CANDIDATE_MATERIALS_AND_MECHANISMS.md) covers spin textures and van der Waals ferromagnets
- [`MATHEMATICS.md`](./MATHEMATICS.md) covers the Hopf invariant, decay, and topological protection
- [`DEFENSIVE_FRAMEWORK.md`](./DEFENSIVE_FRAMEWORK.md) addresses technical critiques

**If you are interested in benchmarks and comparators:**
- [`TARGETS_COMPARATORS_AND_PROJECTIONS.md`](./TARGETS_COMPARATORS_AND_PROJECTIONS.md) contextualizes HELIOS-3D against neuromorphic architectures
- [`OPEN_QUESTIONS.md`](./OPEN_QUESTIONS.md) lists critical uncertainties and blockers

**If you want the long-range vision:**
- [`CORE_ARCHITECTURE.md`](./CORE_ARCHITECTURE.md) describes the MCA + BRC architecture branch
- [`PROPOSED_FABRICATION_PATH_AND_CONTROL.md`](./PROPOSED_FABRICATION_PATH_AND_CONTROL.md) describes the DISH/TPP/ALD fabrication branch

## Understanding Claims

Every claim in this repository is tagged with one of four labels:

| Tag | Meaning |
|---|---|
| `[DEMONSTRATED]` | Verifiable in peer-reviewed literature |
| `[INFERRED]` | Plausible extrapolation from established principles |
| `[PROPOSED]` | Specific architectural integration suggested by HELIOS-3D |
| `[SPECULATIVE]` | Theoretical target or unverified projection |

See [`CLAIMS_MATRIX.md`](./CLAIMS_MATRIX.md) for promotion/demotion criteria.

## What to Read First

The repository is structured so that **near-term paths come before long-range speculation**. If you only read three documents, make them:

1. `README.md` -- status and framing
2. `ALTERNATIVE_MATERIALS_AND_METHODS.md` -- the current baseline
3. `CLAIMS_MATRIX.md` -- evidence discipline

Everything else branches from there.
