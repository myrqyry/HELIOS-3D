# ☀️ HELIOS-3D: Research Documentation

!!! warning "Staging Note"
    This document collection includes both near-term baseline paths and speculative **long-range branches** of HELIOS-3D.
    The current minimum credible path is documented in [ALTERNATIVE_MATERIALS_AND_METHODS.md](./ALTERNATIVE_MATERIALS_AND_METHODS.md).

> A proposed spintronic coprocessor architecture leveraging skyrmions and 3D hopfions to investigate sub-Landauer thermodynamic limits.

HELIOS-3D is a staged research hypothesis. The current minimum credible path is planar-first and electrically read; the hopfion, IFE, and freeform 3D-vdW fabrication story is a later branch.

## ⚡ Project Goals (Hypothesized)
- **Efficiency:** Investigation of low-energy switching and readout regimes, starting from experimentally credible planar devices.
- **Density:** 3D volumetric scaling as a later-stage hypothesis, not a prerequisite for the first demonstrator.
- **Status:** Conceptual / Research Phase
