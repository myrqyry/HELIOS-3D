# References

This document consolidates the peer-reviewed literature, preprints, and contextual sources cited across the HELIOS-3D repository. Sources are grouped by topic and tagged with their relevance to the project.

- **Room-temperature single-layer 2D van der Waals ferromagnet hosting skyrmions.** (2024). [Demonstrates Curie temperature for Fe3GaTe2 on flat substrates.] `[DEMONSTRATED]`
- **Magnetization switching in Fe3GaTe2 via orbital torque at ambient temperatures.** [Demonstrates room-temperature control.] `[DEMONSTRATED]`

### Altermagnets and Compensated Ferrimagnets
- **Spin Current Generation Controlled by the Neel State in a Compensated Ferrimagnet.** arXiv:2507.05618 (2025/2026). [Demonstrates Neel-state-controlled spin pumping as a readout mechanism.] `[INFERRED]`
- **Current-driven nonlinear skyrmion dynamics in altermagnets.** arXiv:2601.13499 (2026). [Demonstrates current-driven helicity motion and THz-scale nonlinear mode mixing.] `[INFERRED]`

### Twistronics / Super-Moire
- **Super-moire spin textures from twisted van der Waals ferromagnetic layers.** [Demonstrates 1.1-degree twist yielding Neel-type skyrmions in CrI3.] `[DEMONSTRATED]`

## Thermodynamic Computing

- **Nonlinear thermodynamic computing out of equilibrium.** (2024). [Demonstrates noise-powered logic in nano-scale reservoirs.] `[DEMONSTRATED]`
