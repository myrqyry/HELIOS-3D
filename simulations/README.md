# HELIOS-3D Simulations

This directory contains micromagnetic scripts for the Landau-Lifshitz-Gilbert (LLG) equations.

It provides open-source configurations for solvers like **MuMax3**, **OOMMF**, and **MagneX** to allow the community to replicate hopfion dynamics.

> ⚠️ **Note:** Some simulation files are currently placeholders or rudimentary setups intended to be expanded into full challenge tasks. See individual files and `BENCHMARKS.md` for current status.
